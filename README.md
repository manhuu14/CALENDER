# 📅 Interactive Calendar Component

A polished and responsive **Interactive Calendar Component** built using React / Next.js.
This project is inspired by a physical wall calendar design and focuses on delivering a clean UI, smooth user experience, and practical functionality.

---

## 🚀 Features

* 🖼 **Wall Calendar Aesthetic**

  * Clean layout inspired by a physical wall calendar
  * Hero image integrated with the monthly calendar grid

* 📆 **Day Range Selection**

  * Select start and end dates
  * Highlighted range between selected dates
  * Clear visual states for start, end, and in-between days

* 📝 **Integrated Notes Section**

  * Add notes for selected dates or date ranges
  * Useful for reminders and planning

* 📱 **Fully Responsive Design**

  * Desktop: Side-by-side layout (image + calendar + notes)
  * Mobile: Stacked layout for smooth usability

* 💾 **Client-Side Persistence**

  * Uses localStorage to save selected dates and notes

---

* React / Next.js
* JavaScript / TypeScript
* CSS / Tailwind (or your styling choice)

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/manhuu14/CALLENDER.git
cd CALLENDER
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:3000
```
